The root `app.js` (declared as `main`) is the sole runtime: it loads shared dependencies declared in this top-level `package.json`, starts the Express application from `app_core`, mounts the domain controllers from `domain_listings_reviews`, serves EJS views from `views_templates`, and proxies static files from `frontend_assets`. All four children depend on the same Express/Mongoose/Passport stack defined here; there is no inter-child package — communication flows through the mounted route tree and shared middleware exposed by `app_core`.
